Mastermind is a code-breaking game typically played with a decoding board and colored pegs. One player, the code maker, secretly arranges a sequence of four colored pegs behind a shield, which becomes the secret code. The other player, the code breaker, has a set number of attempts to guess the code. For each attempt, the code breaker arranges a row of pegs on the board. After each guess, the code maker provides feedback using small pegs: a black peg indicates a correct color in the correct position, while a white peg indicates a correct color in the wrong position. No peg means the color is not in the code. The game continues until the code breaker correctly guesses the code or runs out of attempts.
