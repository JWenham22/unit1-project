Mastermind

Mastermind is a code-breaking game where the objective is to guess a secret code made up of 4 colors. The code breaker (you) has 10 attempts to crack the code by selecting colors from a palette and placing them in slots on the current row. After submitting a guess, feedback is provided: black pegs indicate correct colors in the correct positions, while white pegs indicate correct colors in the wrong positions. Use this feedback to refine your guesses. The game is won by guessing the exact code (4 black pegs) within 10 attempts; otherwise, the secret code is revealed.
